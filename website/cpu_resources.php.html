editor

I personally use notepad++ for this.
Notepad++ download: their official website
Notepad++ stylesheet: for GR8CPU Rev2.1
To import the stylesheet into notepad++:

  1. go to language -> user defined language -> define your language
  2. click import
  3. select the stylesheet file you downloaded
  4. still in the language definer, click "save as..." and name it something line "assembly GR8CPU Rev2.1"
