programming GR8CPU Rev2.1 in assembly

First, you must read the technical description of GR8CPU Rev2.1.
This is by far the most important information, because you need to know a bit of behind-the-scenes to program assembly.

Next, may want to read read the guide to using my assembler.
This shows the general structure of an assembly program, and is also very helpful. You may also skip this, but i recommend reading it.

Finally, i advise you to go to my online CPU control, as you can easily test all sorts of programs here without installing anything.
Simply click on the "hello world" example and program away.

If you do want to install software, you need to go to my CPU resources page.
It will contain all the resources needed to try this on your own (not yet done).
