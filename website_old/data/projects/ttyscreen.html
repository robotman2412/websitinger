The CPU itself is currently finished.
But it's peripherals are most certainly not.
Because we need a screen!
And a controller.

The screen is a bit of a challenge, but it should be do-able.
It emulates 800x600 VGA, but the output signal is actually less then 200x150.
Why? Well, because the pixel clock otherwise needed would be 40MHz, wayyy too fast to be reliable on a breadboard.
Since it is a text display, i should tell you the resolution in characters also. It is 18 rows by 22 colomns.
